/entities

A collection of entities (objects with geolocation) describing places, landmarks and points of interest

/entities get post

get

Get all entities, optionally filtered

post

Create a new entity

get /entities

Get all entities, optionally filtered

  • Request
  • Response

Headers

  • Authorization: required (string)

    The auth token for this request

    Example:

    0da0055c-e606-4e27-b888-bcfefce60d86

Query Parameters

  • start: (integer)

    The first page to return

  • pages: (integer)

    The number of pages to return

  • lat: (number)

    Latitude

  • long: (number)

    Longitude

  • radius: (number)

    An epsilon which specifies the radius to be searched

HTTP status code 200

OK

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/draft-03/schema",
  "type": "object",
  "description": "A collection of entities",
  "properties": {
    "size":  { "type": "integer", "required": true },
      "entities":  {
        "type": "array",
        "items": { "$ref": "entity" }
      }
  }
}

  "items" :
